\# Context Engine



\## Purpose



The Context Engine is responsible for building the current operating context for an AI executive.



It answers one question:



> \*\*What is happening right now?\*\*



Unlike the Knowledge Engine, which provides long-term company knowledge, the Context Engine provides dynamic information that changes continuously.



\---



\## Responsibilities



\- Collect current operational information

\- Aggregate context from multiple systems

\- Prioritize relevant context

\- Eliminate duplicate information

\- Produce a unified Executive Context Package



\---



\## Context Sources



\- Founder Requests

\- Notion

\- GitHub

\- Executive Reports

\- Company Metrics

\- Product Telemetry

\- Calendar

\- Meetings



\---



\## Outputs



\- Executive Context Package

\- Context Metadata

\- Context Priority

\- Context Timestamp
