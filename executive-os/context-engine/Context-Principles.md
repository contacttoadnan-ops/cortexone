\# Context Principles



\## Principle 1



Context represents the current state of the business.



\---



\## Principle 2



Context is temporary.



It exists only for the current execution.



\---



\## Principle 3



Context never replaces company knowledge.



\---



\## Principle 4



The newest verified information has priority.



\---



\## Principle 5



Context must identify its source.



\---



\## Principle 6



Conflicting context must be flagged.



\---



\## Principle 7



Only relevant context is delivered to an executive.



\---



\## Principle 8



Context expires after execution.



\---



\## Principle 9



Every context package includes a timestamp.



\---



\## Principle 10



No executive may make decisions without context.
