\# Memory Principles



\## Principle 1



Memory is persistent.



\---



\## Principle 2



Only valuable information becomes memory.



\---



\## Principle 3



Temporary context never becomes memory automatically.



\---



\## Principle 4



Founder preferences have the highest priority.



\---



\## Principle 5



Every memory has an owner.



\---



\## Principle 6



Every memory has a source.



\---



\## Principle 7



Memories must be traceable.



\---



\## Principle 8



Duplicate memories are merged.



\---



\## Principle 9



Obsolete memories are archived.



\---



\## Principle 10



Memory improves executive performance over time.
