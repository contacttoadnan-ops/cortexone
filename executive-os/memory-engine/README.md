\# Memory Engine



\## Purpose



The Memory Engine manages persistent organizational memory for AI executives.



It answers one question:



> \*\*What should I remember?\*\*



Unlike the Context Engine, which is temporary, the Memory Engine stores information that should persist across executions.



\---



\## Responsibilities



\- Store long-term executive memory

\- Retrieve relevant memories

\- Organize memories by type

\- Preserve founder preferences

\- Record important decisions

\- Archive obsolete memories

\- Prevent duplicate memories



\---



\## Memory Categories



\- Founder Preferences

\- Executive Decisions

\- Lessons Learned

\- Customer Knowledge

\- Operational History

\- Company Milestones

\- Strategic Decisions



\---



\## Outputs



\- Executive Memory Package

\- Relevant Memories

\- Memory Metadata
